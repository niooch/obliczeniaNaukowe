# Notatki do Listy 4.
Na liście bedziemy rozwiązywać układy równań macierzy trójdiagonalnych (w tej macierzy są macierze).
## Zadania
Nalezy zaproponowac algorytm rozwiązujący układ równań macierzy trójdiagonalnych (w tej macierzy są macierze) i zaimplementować go w języku Julia, o złożoności czasowej O(n), gdzie n to liczba macierzy na przekątnej głównej. Rozklad LU lub eliminacja Gaussa mogą być pomocne. (poprzednie wyklady -- dzisaj jest 09.12.2025)
Rowniez napisac tesy prownujace nasz algorytm z algorytmem ogólnym (np. z pakietu LinearAlgebra w Julii).
Wymagane sa rowniez procedury czytajace macierze trójdiagonalne z pliku i zapisujace wynik do pliku: bedziemy znac rozwiazanie dokladne, nalezy policzyc blad wzgledny w normie 2.
